Carpeta para alojar documentación de interés para el proyecto. Puede usarse para cualquier tipo de documentación, instrucciones, manuales, etc.
Usad nombres descriptivos. 
Ficheros que contiene:
 -httpd-xampp.conf: Contiene la configuración que utilizamos en Apache.
 -configuracionApache: documento explicativo de los pasos seguidos para configurar nuestro Apache.
